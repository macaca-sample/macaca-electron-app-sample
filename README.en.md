# macaca-electron-app-sample

---

## run

```bash
$ npm i
$ npm run build
$ npm run dist
```

Select dmg, mac.zip or exe installation in dist directory, and then find the installation path to replace the binary parameter under the chromeOptions object of sample.test.js

- Mac:

```bash
$ npm run mac-start
```

```bash
$ npm run test
```

- Windows:

```bash
$ npm run win-start
```

```bash
$ npm run win-server
```

```bash
$ npm run test
```
